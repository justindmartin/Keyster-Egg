Keyster Egg
=================

The KeysterEgg class lets you make key combination easter eggs in JavaScript, just like Konami used to make.

## Usage Example

```Javascript
var egg = new KeysterEgg({
  callback: function(){
    alert('Hi there!');
  },
  keyCombo: 'hello'
});
```

## Use the following for directional keys in KeysterEgg combinations
↑  equals  UP

↓  equals  DOWN

←  equals  LEFT

→  equals  RIGHT
